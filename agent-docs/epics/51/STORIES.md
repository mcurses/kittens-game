# Epic 51 — Science Panels Parity

Close science ❌ items: policy panel, metaphysics panel, flavor text.

## Stories

### 51-01 Policy Panel
**Goal:** Add a dedicated policies panel accessible from the science tab.

**ACs:**
- [ ] Policy panel shows available policies (unlocked by techs)
- [ ] Each policy shows: name, description, adopt/block status
- [ ] Adopt/block toggle dispatches engine action
- [ ] Panel visible only after relevant tech unlocked
- [ ] Test: verify policy panel renders and toggle works

### 51-02 Metaphysics / Prestige Perks Panel
**Goal:** Add a panel for prestige perks, visible after Metaphysics is researched.

**ACs:**
- [ ] Panel lists all metaphysics perks with name, description, cost
- [ ] Buy button for unowned perks, "Owned" badge for purchased
- [ ] Paragon cost display
- [ ] Panel visible only after Metaphysics researched
- [ ] Test: verify perk list renders and purchase works

### 51-03 Science Flavor Text
**Goal:** Add flavor text to technology items in the inspector.

**ACs:**
- [ ] Define flavor text map for technologies
- [ ] Inspector shows flavor text when hovering a tech
- [ ] Test: verify at least 3 techs show flavor text
