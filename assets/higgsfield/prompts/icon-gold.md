# icon-gold
Pulls from [_icon-style.md](_icon-style.md).

**Backdrop color**: `--res-gold` (#D9A441), warm sunlit gold disc. Gold is
one of 24 resources that today shows only as a TEXT label in the UI — this
icon is part of the first wave that closes those glyph gaps.

**Subject**: three chunky chibi gold coins stacked in a tight pile (bottom
coin half-visible behind, two more stacked centered above), each shown
edge-on with a clear circular outline. The pile takes the central 60% of
the disc, centered horizontally, slightly skewed so the top coin tilts ~10°
right to suggest a casual stack.

**Accent**: one warm-cream (#F2E6CF) highlight pixel on the upper-left edge
of the topmost coin to suggest light catching the metal.

**Aspect**: 1:1. Pipeline: 512² raw → `sips -Z 128` → `cwebp -lossless` →
`exports/icons/gold.webp`.

**Belongs to**: resource (`gold`).
