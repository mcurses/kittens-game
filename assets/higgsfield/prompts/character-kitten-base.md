# character-kitten-base

**Subject**: A chibi pixel-art kitten portrait — front-facing, looking slightly off-camera, large eyes, soft round body. Single character against a soft parchment-toned background that suggests a frame. This is the GENERIC base portrait that will later be specialized into trait/role variants.

**Style anchors**: see [STYLE-DNA.md](../STYLE-DNA.md). Pixel art, character-portrait composition (subject fills ~70% of frame), readable at small to medium sizes.

**Palette anchors**:
- backdrop: `--bg #F2E6CF` with subtle `--surface-2 #E2D0AC` vignette
- cat body: warm tabby palette — cream `--bg` with `--accent #C97D5D` stripes, white paw socks
- eyes: large round, `--res-science #7A9DC2` (blue) — distinctive but not unnatural
- ink: `--ink #2B221A` (outlines, pupils, mouth)
- inner ears + nose: `--res-faith #C23574` (sacred magenta — small accent)

**Negative**: see STYLE-DNA. No human features (no clothing here — this is the BASE), no accessories yet, no anime style, no realistic cat photo. The portrait should read as personable but neutral so traits/roles can layer in.

**Aspect**: 1:1 (`character` variant). 1024×1024 raw → 512×512 final WEBP at `exports/characters/kitten-base.webp`.

**Belongs to loop beats**: `settlement` onwards — kitten characters become visible.
